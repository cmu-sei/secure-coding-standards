# Introduction to Java Recommendations

The [Java Coding Guidelines](http://www.cert.org/secure-coding/publications/books/java-coding-guidelines.cfm) book was published in hard copy in September 2013. Not all of the guidelines are applicable to the Android environment. Our analysis of Android applicability of the Java coding guidelines is [here](/android-secure-coding-standard/by-language/java-coding-language/applicable-in-principle-to-android-java-rulesrecomendations) . The analysis is presented in a chart form. For each Java guideline found to require Android-specific implementation advice, that advice is provided in a column of the analysis chart.
