# Android (DRD)

The following rules and guidelines are specific only to the Android platform. These do not apply to the development of Java or C programs for non-Android platforms. (The full set of Android-relevant rules and guidelines are [here](/android-secure-coding-standard/by-language/android-only) .) The term *sensitive* incorporates the Java glossary definition of *[sensitive data](/sei-cert-oracle-coding-standard-for-java/back-matter/rule-bb-glossary)* as well as the Android concept of permission-protected.
