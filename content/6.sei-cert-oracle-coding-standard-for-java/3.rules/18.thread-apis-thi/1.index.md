# Thread APIs (THI)

## Rules

-   [THI00-J. Do not invoke Thread.run()](/sei-cert-oracle-coding-standard-for-java/rules/thread-apis-thi/thi00-j)
-   [THI01-J. Do not invoke ThreadGroup methods](/sei-cert-oracle-coding-standard-for-java/rules/thread-apis-thi/thi01-j)
-   [THI02-J. Notify all waiting threads rather than a single thread](/sei-cert-oracle-coding-standard-for-java/rules/thread-apis-thi/thi02-j)
-   [THI03-J. Always invoke wait() and await() methods inside a loop](/sei-cert-oracle-coding-standard-for-java/rules/thread-apis-thi/thi03-j)
-   [THI04-J. Ensure that threads performing blocking operations can be terminated](/sei-cert-oracle-coding-standard-for-java/rules/thread-apis-thi/thi04-j)
-   [THI05-J. Do not use Thread.stop() to terminate threads](/sei-cert-oracle-coding-standard-for-java/rules/thread-apis-thi/thi05-j)
