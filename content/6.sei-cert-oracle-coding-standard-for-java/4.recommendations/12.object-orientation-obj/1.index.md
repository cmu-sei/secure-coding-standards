# Object Orientation (OBJ)

-   [OBJ50-J. Never confuse the immutability of a reference with that of the referenced object](/sei-cert-oracle-coding-standard-for-java/recommendations/object-orientation-obj/obj50-j)
-   [OBJ51-J. Minimize the accessibility of classes and their members](/sei-cert-oracle-coding-standard-for-java/recommendations/object-orientation-obj/obj51-j)
-   [OBJ52-J. Write garbage-collection-friendly code](/sei-cert-oracle-coding-standard-for-java/recommendations/object-orientation-obj/obj52-j)
-   [OBJ53-J. Do not use direct buffers for short-lived, infrequently used objects](/sei-cert-oracle-coding-standard-for-java/recommendations/object-orientation-obj/obj53-j)
-   [OBJ54-J. Do not attempt to help the garbage collector by setting local reference variables to null](/sei-cert-oracle-coding-standard-for-java/recommendations/object-orientation-obj/obj54-j)
-   [OBJ55-J. Remove short-lived objects from long-lived container objects](/sei-cert-oracle-coding-standard-for-java/recommendations/object-orientation-obj/obj55-j)
