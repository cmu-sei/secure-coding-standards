# Tech-edit

For a guideline to go to the next level it should contain the previous label and be approved by the person maintaining that previous label. For example, a guideline with the label `draft` is not ready to be marked as `tech-edit` . A guideline that is marked as only `draft-one` is not ready to be marked as `tech-edit` either. The guideline should generally be made eligible for `tech-edit` only after it has been marked `draft-one` . And likewise, a guideline shouldn't be made eligible for `draft-one` until after it has been marked `draft` .
