# SEI CERT C++ Coding Standard

The C++ rules and recommendations in these pages are a work in progress and reflect the current thinking of the secure coding community. Because this is a development website, many pages are incomplete or contain errors. As rules and recommendations mature, they are published in report or book form as official releases. These releases are issued as dictated by the needs and interests of the secure software development community.

The CERT C++ Coding Standard does not currently expose any recommendations.

## Comments and Contributing

We provide access to the community members to contribute after subject matter expertise is verified.

For questions and comments about these standards, open a GitHub issue. For anything else, please [submit feedback](https://www.sei.cmu.edu/contact-us/).

## Secure C++ Coding Books and Downloads

<a href="https://resources.sei.cmu.edu/library/asset-view.cfm?assetID=494932"><img src="/attachments/88046682/88480694.png" /></a> The [CERT C++ Coding Standard, 2016 Edition](https://resources.sei.cmu.edu/library/asset-view.cfm?assetID=494932) provides rules to help programmers ensure that their C++ code reduces security flaws by following secure coding best practices. It is downloadable as a PDF. ( [errata](/sei-cert-cpp-coding-standard/errata-for-sei-cert-cpp-coding-standard-2016-edition) )


<a href="https://resources.sei.cmu.edu/library/asset-view.cfm?assetID=454220"><img src="/attachments/87152044/88022020.png" /></a> The CERT C++ Coding Standard references and relies on the CERT C Coding Standard. The [CERT C Coding Standard, 2016 Edition](https://resources.sei.cmu.edu/library/asset-view.cfm?assetID=454220) provides rules to help programmers ensure that their code complies with the new C11 standard and earlier standards, including C99. It is downloadable as a PDF. ( [errata](/sei-cert-c-coding-standard/errata-for-sei-cert-c-coding-standard-2016-edition) )

  
<a href="http://www.informit.com/store/secure-coding-in-c-and-c-plus-plus-9780321822130"><img src="/attachments/88046682/88487064.png" /></a>
[Secure Coding in C and C++](http://www.informit.com/store/secure-coding-in-c-and-c-plus-plus-9780321822130) identifies the root causes of today's most widespread software vulnerabilities, shows how they can be exploited, reviews the potential consequences, and presents secure alternatives.
