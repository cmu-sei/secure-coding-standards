# Exceptions and Error Handling (ERR)

<div class="contentLayout2">

<div class="columnLayout two-right-sidebar" layout="two-right-sidebar">

<div class="cell normal" data-type="normal">

<div class="innerCell">

-   [ERR50-CPP. Do not abruptly terminate the program](/sei-cert-cpp-coding-standard/rules/exceptions-and-error-handling-err/err50-cpp)
-   [ERR51-CPP. Handle all exceptions](/sei-cert-cpp-coding-standard/rules/exceptions-and-error-handling-err/err51-cpp)
-   [ERR52-CPP. Do not use setjmp() or longjmp()](/sei-cert-cpp-coding-standard/rules/exceptions-and-error-handling-err/err52-cpp)
-   [ERR53-CPP. Do not reference base classes or class data members in a constructor or destructor function-try-block handler](/sei-cert-cpp-coding-standard/rules/exceptions-and-error-handling-err/err53-cpp)
-   [ERR54-CPP. Catch handlers should order their parameter types from most derived to least derived](/sei-cert-cpp-coding-standard/rules/exceptions-and-error-handling-err/err54-cpp)
-   [ERR55-CPP. Honor exception specifications](/sei-cert-cpp-coding-standard/rules/exceptions-and-error-handling-err/err55-cpp)
-   [ERR56-CPP. Guarantee exception safety](/sei-cert-cpp-coding-standard/rules/exceptions-and-error-handling-err/err56-cpp)
-   [ERR57-CPP. Do not leak resources when handling exceptions](/sei-cert-cpp-coding-standard/rules/exceptions-and-error-handling-err/err57-cpp)
-   [ERR58-CPP. Handle all exceptions thrown before main() begins executing](/sei-cert-cpp-coding-standard/rules/exceptions-and-error-handling-err/err58-cpp)
-   [ERR59-CPP. Do not throw an exception across execution boundaries](/sei-cert-cpp-coding-standard/rules/exceptions-and-error-handling-err/err59-cpp)
-   [ERR60-CPP. Exception objects must be nothrow copy constructible](/sei-cert-cpp-coding-standard/rules/exceptions-and-error-handling-err/err60-cpp)
-   [ERR61-CPP. Catch exceptions by lvalue reference](/sei-cert-cpp-coding-standard/rules/exceptions-and-error-handling-err/err61-cpp)
-   [ERR62-CPP. Detect errors when converting a string to a number](/sei-cert-cpp-coding-standard/rules/exceptions-and-error-handling-err/err62-cpp)

  

The following rules from the [SEI CERT C Coding Standard](/sei-cert-c-coding-standard/) also apply in C++:

-   [ERR30-C. Take care when reading errno](/sei-cert-c-coding-standard/rules/error-handling-err/err30-c)
-   [ERR32-C. Do not rely on indeterminate values of errno](/sei-cert-c-coding-standard/rules/error-handling-err/err32-c)
-   [ERR33-C. Detect and handle standard library errors](/sei-cert-c-coding-standard/rules/error-handling-err/err33-c)
-   [ERR34-C. Detect errors when converting a string to a number](/sei-cert-c-coding-standard/rules/error-handling-err/err34-c)
