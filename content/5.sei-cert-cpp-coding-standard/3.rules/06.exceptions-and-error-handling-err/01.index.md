-   [ERR52-CPP. Do not use setjmp() or longjmp()](/sei-cert-cpp-coding-standard/rules/exceptions-and-error-handling-err/err52-cpp)
-   [ERR53-CPP. Do not reference base classes or class data members in a constructor or destructor function-try-block handler](/sei-cert-cpp-coding-standard/rules/exceptions-and-error-handling-err/err53-cpp)

-   [ERR30-C. Take care when reading errno](/sei-cert-c-coding-standard/rules/error-handling-err/err30-c)
-   [ERR32-C. Do not rely on indeterminate values of errno](/sei-cert-c-coding-standard/rules/error-handling-err/err32-c)
-   [ERR33-C. Detect and handle standard library errors](/sei-cert-c-coding-standard/rules/error-handling-err/err33-c)
-   [ERR34-C. Detect errors when converting a string to a number](/sei-cert-c-coding-standard/rules/error-handling-err/err34-c)
