The following rules from the [SEI CERT C Coding Standard](/sei-cert-c-coding-standard/) also apply in C++:

-   [MSC30-C. Do not use the rand() function for generating pseudorandom numbers](/sei-cert-c-coding-standard/rules/miscellaneous-msc/msc30-c)

-   [MSC33-C. Do not pass invalid data to the asctime() function](/sei-cert-c-coding-standard/rules/miscellaneous-msc/msc33-c)
-   [MSC37-C. Ensure that control never reaches the end of a non-void function](/sei-cert-c-coding-standard/rules/miscellaneous-msc/msc37-c)
-   [MSC38-C. Do not treat a predefined identifier as an object if it might only be implemented as a macro](/sei-cert-c-coding-standard/rules/miscellaneous-msc/msc38-c)
-   [MSC39-C. Do not call va_arg() on a va_list that has an indeterminate value](/sei-cert-c-coding-standard/rules/miscellaneous-msc/msc39-c)
-   [MSC40-C. Do not violate constraints](/sei-cert-c-coding-standard/rules/miscellaneous-msc/msc40-c)
-   [MSC41-C. Never hard code sensitive information](/sei-cert-c-coding-standard/rules/miscellaneous-msc/msc41-c)
