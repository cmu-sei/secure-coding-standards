The following rules from the [SEI CERT C Coding Standard](/sei-cert-c-coding-standard/) also apply in C++:

-   [MEM30-C. Do not access freed memory](/sei-cert-c-coding-standard/rules/memory-management-mem/mem30-c)
-   [MEM31-C. Free dynamically allocated memory when no longer needed](/sei-cert-c-coding-standard/rules/memory-management-mem/mem31-c)
-   [MEM34-C. Only free memory allocated dynamically](/sei-cert-c-coding-standard/rules/memory-management-mem/mem34-c)
-   [MEM35-C. Allocate sufficient memory for an object](/sei-cert-c-coding-standard/rules/memory-management-mem/mem35-c)
-   [MEM36-C. Do not modify the alignment of objects by calling realloc()](/sei-cert-c-coding-standard/rules/memory-management-mem/mem36-c)
