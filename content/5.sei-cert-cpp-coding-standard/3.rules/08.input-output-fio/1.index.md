# Input Output (FIO)

<div class="contentLayout2">

<div class="columnLayout two-right-sidebar" layout="two-right-sidebar">

<div class="cell normal" data-type="normal">

<div class="innerCell">

-   [FIO50-CPP. Do not alternately input and output from a file stream without an intervening positioning call](/sei-cert-cpp-coding-standard/rules/input-output-fio/fio50-cpp)
-   [FIO51-CPP. Close files when they are no longer needed](/sei-cert-cpp-coding-standard/rules/input-output-fio/fio51-cpp)

  

The following rules from the [SEI CERT C Coding Standard](/sei-cert-c-coding-standard/) also apply in C++:

-   [FIO30-C. Exclude user input from format strings](/sei-cert-c-coding-standard/rules/input-output-fio/fio30-c)
-   [FIO32-C. Do not perform operations on devices that are only appropriate for files](/sei-cert-c-coding-standard/rules/input-output-fio/fio32-c)
-   [FIO34-C. Distinguish between characters read from a file and EOF or WEOF](/sei-cert-c-coding-standard/rules/input-output-fio/fio34-c)
-   [FIO37-C. Do not assume that fgets() or fgetws() returns a nonempty string when successful](/sei-cert-c-coding-standard/rules/input-output-fio/fio37-c)
-   [FIO38-C. Do not copy a FILE object](/sei-cert-c-coding-standard/rules/input-output-fio/fio38-c)
-   [FIO39-C. Do not alternately input and output from a stream without an intervening flush or positioning call](/sei-cert-c-coding-standard/rules/input-output-fio/fio39-c)
-   [FIO40-C. Reset strings on fgets() or fgetws() failure](/sei-cert-c-coding-standard/rules/input-output-fio/fio40-c)
-   [FIO41-C. Do not call getc(), putc(), getwc(), or putwc() with a stream argument that has side effects](/sei-cert-c-coding-standard/rules/input-output-fio/fio41-c)
-   [FIO42-C. Close files when they are no longer needed](/sei-cert-c-coding-standard/rules/input-output-fio/fio42-c)
-   [FIO44-C. Only use values for fsetpos() that are returned from fgetpos()](/sei-cert-c-coding-standard/rules/input-output-fio/fio44-c)
-   [FIO45-C. Avoid TOCTOU race conditions while accessing files](/sei-cert-c-coding-standard/rules/input-output-fio/fio45-c)
-   [FIO46-C. Do not access a closed file](/sei-cert-c-coding-standard/rules/input-output-fio/fio46-c)
-   [FIO47-C. Use valid format strings](/sei-cert-c-coding-standard/rules/input-output-fio/fio47-c)
