The following rules from the [SEI CERT C Coding Standard](/sei-cert-c-coding-standard/) also apply in C++:

-   [STR30-C. Do not attempt to modify string literals](/sei-cert-c-coding-standard/rules/characters-and-strings-str/str30-c)

-   [STR34-C. Cast characters to unsigned char before converting to larger integer sizes](/sei-cert-c-coding-standard/rules/characters-and-strings-str/str34-c)
-   [STR37-C. Arguments to character-handling functions must be representable as an unsigned char](/sei-cert-c-coding-standard/rules/characters-and-strings-str/str37-c)
-   [STR38-C. Do not confuse narrow and wide character strings and functions](/sei-cert-c-coding-standard/rules/characters-and-strings-str/str38-c)
